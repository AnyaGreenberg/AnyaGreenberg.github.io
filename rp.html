RepeatProfiler

Project Details

Website

RepeatProfiler is a command line tool designed to study repetitive DNA dynamics using low-coverage, short-read data, especially over short evolutionary times scales in groups with limited genomic resources. The tool maps reads to consensus sequences of repeat(s) of interest, generates visually enhanced read depth profiles for each repeat, and facilitates comparison of profiles within and among samples. There are multiple other functions of the tool including correlation analysis between user-defined groups and preparation of input files for phylogenetic analysis using signals from variants.
